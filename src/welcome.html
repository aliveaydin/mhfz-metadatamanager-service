Welcome to mhfz-metadataManager Service

Manages document metadata (creation, editing, enrichment via DOI/DKB), document classification, supports predefined/custom types, and maintains enhanced schemas for search and compliance. Handles enrichment jobs, links metadata to documents, enables flexible metadata input, and facilitates export/import of document metadata.

Available Routes

  • getDocumentMetadata => GET /documentmetadatas/:documentMetadataId
  • createDocumentMetadata => POST /documentmetadatas
  • updateDocumentMetadata => PATCH /documentmetadatas/:documentMetadataId
  • deleteDocumentMetadata => DELETE /documentmetadatas/:documentMetadataId
  • listDocumentMetadata => GET /documentmetadata
  • getDocumentType => GET /documenttypes/:documentTypeId
  • createDocumentType => POST /documenttypes
  • updateDocumentType => PATCH /documenttypes/:documentTypeId
  • deleteDocumentType => DELETE /documenttypes/:documentTypeId
  • listDocumentTypes => GET /documenttypes
  • getMetadataEnrichmentJob => GET /metadataenrichmentjobs/:metadataEnrichmentJobId
  • createMetadataEnrichmentJob => POST /metadataenrichmentjobs
  • updateMetadataEnrichmentJob => PATCH /metadataenrichmentjobs/:metadataEnrichmentJobId
  • deleteMetadataEnrichmentJob => DELETE /metadataenrichmentjobs/:metadataEnrichmentJobId
  • listMetadataEnrichmentJobs => GET /metadataenrichmentjobs
